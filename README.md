# Resoluções dos Exercícios – Laços / Looping em Python

Este repositório contém as resoluções dos exercícios da lista de **Laços (Looping)** disponível em:

https://www.pythonprogressivo.net/2018/06/Lista-Exercicios-Lacos-Looping-Python.html

---

## 🧠 Exercício 1

Faça um programa que peça uma nota, entre zero e dez. Mostre uma mensagem caso o valor seja inválido e continue pedindo até que o usuário informe um valor válido.

---

## 📖 Exercício 2

Faça um programa que leia um nome de usuário e a sua senha e não aceite a senha igual ao nome do usuário, mostrando uma mensagem de erro e voltando a pedir as informações.
